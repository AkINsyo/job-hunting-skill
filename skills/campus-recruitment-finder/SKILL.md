# 校招官网自动查找 Skill

## 概述

自动搜索指定企业的校园招聘官网，验证链接有效性，生成结构化汇总文档。

## 使用方式

用户提供公司名列表，Skill 自动完成搜索→提取→验证→输出。

## 执行步骤

### Step 1: 解析输入

从用户输入中提取公司名列表。支持的输入格式：
- 逗号分隔：`小米,字节跳动,腾讯`
- 空格分隔：`小米 字节跳动 腾讯`
- 换行分隔
- 或用户说"帮我找以下公司的校招网站：xxx"

如果用户未指定公司，使用默认列表（见下方）。

### Step 2: 搜索校招链接

对每家公司，使用 `web_search` 搜索以下关键词（按优先级）：

1. `{公司名} 校招官网 2025 2026`
2. `{公司名} campus recruitment official`
3. `{公司名} 校园招聘`

**提取规则**：
- 优先提取官方域名（如 `hr.xiaomi.com`, `jobs.bytedance.com`）
- 排除第三方平台（牛客、知乎、CSDN 等）的结果
- 如果搜索结果中有 `campus`、`recruit`、`join` 等路径关键词，优先采用

**并行搜索**：使用 `delegate_task` 的 tasks 批量模式，并行搜索 3 家公司以提高效率。

### Step 3: 验证链接可访问性

对提取到的 URL，使用 `terminal` 执行 curl 验证：

```bash
curl -sL -o /dev/null -w "%{http_code}" --max-time 10 "URL"
```

- `200` → ✅ 可访问
- `301/302` → ⚠️ 重定向（记录跳转目标）
- `403/404/timeout` → ❌ 不可访问

### Step 4: 生成输出

将结果写入 Markdown 文件，格式如下：

```markdown
# 🎓 校招官网汇总

> 自动搜索生成于：{日期}

| 公司 | 校招官网 | 状态 | 搜索关键词 |
|------|---------|------|-----------|
| 小米 | https://... | ✅ | 小米校招官网 |
| ... | ... | ... | ... |

## ⚠️ 注意
- 链接有效性基于搜索时的检测，校招季页面可能更新
- 建议点击链接二次确认
```

保存路径：当前工作目录下 `校招网站汇总_{日期}.md`

### Step 5: 输出结果

向用户展示汇总表格，并告知文件保存位置。

## Pitfalls

1. **URL 时效性**：校招页面每年更换路径，检测结果仅代表当前状态
2. **搜索干扰**：搜索引擎结果中大量第三方平台（牛客、知乎），需严格过滤
3. **反爬限制**：部分校招网站有反爬机制，curl 可能返回非 200 但实际可访问
4. **并行限制**：delegate_task 最多 3 个并行任务，超过需分批执行
5. **网络环境**：当前环境可能无法访问 Google/Bing，优先使用可工作的搜索引擎
